# 🍩 Donat Admin WA Bot

Bot WhatsApp berbasis Node.js yang digunakan untuk membantu administrasi toko donat, seperti membalas pesan pelanggan, mengelola pesanan, dan otomatisasi tugas dengan bantuan AI.

## 🚧 Status

Project ini masih dalam tahap pengembangan (belum 100% selesai).

## ✨ Fitur (sementara)

- Auto reply WhatsApp menggunakan AI
- Sistem ON/OFF bot
- Respon chat pelanggan otomatis
- Integrasi API AI

## 🛠️ Teknologi

- Node.js
- Baileys (WhatsApp Web API)
- AI API (Gemini / lainnya)
- MongoDB

## 📦 Instalasi (sementara)

```bash
git clone https://github.com/username/donat-admin-wa-bot.git
cd donat-admin-wa-bot
npm install
npm run start
```
